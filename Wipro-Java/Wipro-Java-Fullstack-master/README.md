# Wipro NGA: Full-Stack Transformation 🚀

## 📌 Overview
This repository contains the technical journey and project work completed during the **Wipro NGA Training Program** (April – June 2026). The program is a deep-dive into full-stack engineering, covering the modern Java ecosystem, cloud-native architectures, and Generative AI integration [cite: 8, 11].

## 🏗️ Program Structure
The training is structured into four distinct phases, transitioning from core logic to complex distributed systems:

### **Phase 1: Core Java & Java Fundamentals**
* **Git & Maven:** Repository management, build lifecycles, and dependency management using POM structure [cite: 8].
* **Java Mastery:** Deep dive into Java architecture, Java 8-23 features (Records, Virtual Threads), OOPs, Collections, and Concurrency [cite: 8].
* **Quality Assurance:** Unit testing with JUnit 5 and code coverage tools [cite: 8].
* my progress from **Day 1 to Day 11** of the training, specifically focusing on the Java mastery required for the first program milestone [cite: 8].
